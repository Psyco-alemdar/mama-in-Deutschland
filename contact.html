Important

Le formulaire de contact ne remplace pas les urgences médicales, les conseils juridiques, les institutions officielles ou une consultation personnalisée.

Pour une urgence : 112. Pour un problème médical urgent mais non vital : 116117. Pour grossesse/bébé : Frauenarzt, Hebamme, Kinderarzt ou clinique.

Formulaire de contact

Le message est envoyé vers Supabase si la table contact_messages existe. Sinon il est sauvegardé localement et tu peux le copier.

À qui s’adresser ?

Pour un problème technique de plateforme, le formulaire est adapté. Pour santé, documents officiels ou urgence, contacte les services correspondants.
